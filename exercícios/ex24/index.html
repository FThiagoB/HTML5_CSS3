iframes - Começando os estudos sobre quadros em linha

Criamos iframes usando a tag <iframe></iframe>. O conteúdo interno da tag será mostrado caso os iframes não sejam suportados pelo navegador.

O tamanho padrão dos iframes é 300x150, o que podemos modificar na própria tag ou usando um estilo CSS. Diferente das outras tags, ao especificar os dois tipos de configuração, o estilo especificado no CSS será soberano.
