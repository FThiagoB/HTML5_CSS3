iframes - conteúdo dinâmico

Para modificar o conteúdo de dado iframe, podemos especificar o parâmetro target de um link com o valor do parâmetro name do iframe. Não apenas páginas podem ser especificadas, como imagens e vídeos também funcionam. Caso tenha acabado de olhar, volte para a página anterior.

Selecione uma das escolhas abaixo e veja o que acontece:

  • Página do projeto [Github]
  • Tabela com rolagem
  • Página interna
  • Uma imagem qualquer
