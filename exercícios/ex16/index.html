Representando cores em CSS

O CSS permite que a representação de cores de diferentes maneiras. Podemos citar:

  • Utilizando o nome da cor. O CSS reconhece quando especificamos uma cor escrevendo seu nome, isso funciona bem para algumas cores gerais mais isso não é adequado quando queremos um tom em particular ou quando queremos modificar a transparência da cor usada. Existem em torno de 140 cores nomeadas, que podem ser vistas aqui;
  • Usando um hexadecimal RGB. Dessa forma podemos definir exatamente a tonalidade da cor desejada seguinto o seguinte formato: #RRGGBB;
  • Usando um decimal RGB. Da mesma forma que a forma anterior, podemos usar o padrão RGB só que dessa vez especificando usando decimais. Usamos a função rgb(RED, GREEN, BLUE);
  • Usando o código HSL. Isso especifica as características de matiz, saturação e iluminação da cor desejada;
  • Usando RGB com alpha. Adicionando um parâmetro extra ao formato RGB podemo definir a transparência, também chamado de alpha, da cor. Isso pode ser feito usando hexadecimal, #RRGGBBAA, ou decimal, rgba(RED, GREEN, BLUE, ALPHA).

As páginas abaixo exibem outras funcionalidades usando cores.

  • gradiente.html: O CSS permite a criação de gradiente, lineares e radiais, permitindo a criação de diversos efeitos;
  • pagina.html: Um dos aspectos importantes ao criar uma página é a seleção adequada da paleta de cores, pois isso dá para a página um aspecto agradável.
