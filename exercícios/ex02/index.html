Usando parágrafos, quebras de linha e emojis

Você pode escrever um Parágrafo de qualquer jeito, basta colocar tudo no meio do par de tags <p> e </p>

Podemos criar quebras de linha usando a tag <br>.
Quebrar linhas no código-fonte não funciona, é necessário usar a tag para especificar!

HTML Entities e emojis

Para representar determinados símbolos e emojis devemos consultar seu código ou, caso exista, seu nome específico.

Para expressar símbolos matemáticos devemos verificar qual seu nome, que geralmente é intuitivo, por exemplo: ∫f(x)dx ou ΔL=αL0ΔT.

Também é possível usar emojis!!!
Para adicionar um emoji podemos simplesmente copiar e colar diretamente no código-fonte (nem sempre funciona) ou usar o código no seguinte formato: &#xCODE;
