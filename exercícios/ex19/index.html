Seletores CSS

Usamos os seletores para especificar quais elementos um determinado conjunto de regras é aplicado. Basicamente, os seguintes seletores podem ser utilizados:

  • tag: seleciona todos os elementos que correspondem a tag fornecida;
  • classe: seleciona todos os elementos que possuem o valor class igual ao nome fornecido. O seletor é usado da seguinte forma: .nome-classe;
  • id: seleciona todos os elementos que possuem o valor id igual ao nome fornecido. O seletor é usado da seguinte forma: #nome-id;
  • universal: usado para selecionar todos os elementos do documento. O caractere * é usado como coringa;
  • atributo: Este seletor irá escolher elementos baseados no valor de um de seus atributos, ou até mesmo pelo próprio atributo. Geralmente assumme a forma [atrib] [atrib=valor] [atrib~=valor] [atrib|=valor] [atrib^=valor] [atrib$=valor] [atrib*=valor]
