Aprendendo sobre formatação de texto.

Uma das grandes mudanças ocorridas do HTML4 para o HTML5 foi o foco maior em semântica do que na forma. Muitas tags existentes na versão 4 foram consideradas obsoletas no HTML5.

Uma pequena lista das tags apresentadas pelo Guanabara é mostrado abaixo.

  • Para usarmos negrito em uma palavra ou frase utilizamos as tags <strong> e </strong>;
  • Para dar enfâse para uma palavra ou frase podemos colocá-la em itálico usando a tag <em> e </em>;
  • Para usar o efeito de marca-texto usamos a tag <mark> e </mark>;
  • Curiosamente a tag <small> ainda existe em HTML5, embora a tag <big> esteja depreciada;
  • Quando não queremos apagar um texto mas apenas indicar que o conteúdo deva ser desconsiderado (geralmente como pensamentos aleatórios do autor ou tradutor), pode-se usar a tag <del> e </del>;
  • Já para indicar que dado texto foi adicionado depois, ou que é necessário prestar mais atenção a determinada parte, usamos a tag <ins> e </ins>;
  • Os subscritos e sobrescritos podem ser adicionados com as tags <sub> e <sup>, respectivamente;
  • Quando desejamos escrever um código fonte podemos usar a tag <code> e </code>
  • Quando desejamos que certo texto seja exibido exatamente como foi digitado, geralmente quando estamos escrevendo um código-fonte, usamos a tag <pre> e </pre>;
  • Para usar citações no meio de um parágrafo podemos usar a tag <q> </q> (vem de quote);
  • Para citações longa, que possuam um parágrafo próprio, usamos a tag <blockquote> e </blockquote>, ao especificar o parâmetro 'cite' podemos colocar um link para a fonte;
  • Uma das novidades do HTML5 é a tag usada para abreviações, que abre um pequeno popup ao passar em cima da palavra com o mouse. Usamos abreviações com as tags <abbr> e </abbr>;
  • Uma tag com funcionalidade estranha é a tag <bdo> e </bdo> (significa bi-directional override) que ao especificar o parâmetro 'dir' com rtl (right-to-left) inverterá o texto especificado.

Podemos combinar tags para produzir alguns efeitos interessantes (embora ainda seja apenas arroz e feijão). O estudo de qualquer assunto é demorado mas se for dado tempo suficiente não é impossível. A diferença entre dificuldade e trabalho também é importante, uma frase interessante diz que o mais difícil é aprender a ler, o resto está escrito. Uma das minhas passagens favoritas de Frankenstein é:

"Por mais que se tenha feito", brandou a alma de Victor Frankenstein, "muito mais eu alcançarei. Desbravarei novos caminhos, explorarei forças desconhecidas e revelarei ao mundo os mistérios da criação."

O uso de equações e expressões matemática são úteis quando o assunto está diretamente relacionado à matemática ou alguma área da ciência. Por exemplo, a primeira lei da termodinâmica pode ser expressa como Q = ΔU + Τ, onde o calor trocado por certo sistema é igual à variação de sua energia interna mais o trabalho realizado ou recebido pelo sistema.
