Usando CSS

Podemos especificar formatações que especificam a maneira como os elementos HTML devem ser exibidos usando CSS. Existem basicamente três formas de fazer isso:

  • Estilos inline, especificado individualmente em cada tag;
  • Estilos especificados na própria página, usando a tag <style>;
  • Usando arquivos de estilo externos, dessa forma podemos especificar o arquivo de estilo no header da página.

Nesta página foi utilizado um arquivo .css para definir o estilo. Isso é bastante útil pois esse mesmo estilo pode ser utilizado em outra página ter o trabalho de digitar tudo de novo. Por exemplo, a página 02 está usando o mesmo arquivo CSS usado aqui.

Podemos perceber que ao especificar formatações mais internas, usando inline ou <style>, podemos sobrescrever os estilos definidos nos arquivos CSS.
