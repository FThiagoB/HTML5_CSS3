Usando imagens de fundo

O CSS permite usarmos imagens e cores de fundo. Podemos aplicar background diferentes em uma mesma página. A lista abaixo enumera algumas propriedades referentes ao background que podemos configurar.

  • background-color: Configura uma cor de fundo para o elemento;
  • background-image: Configura uma ou mais imagens de fundo para o elemento;
  • background-position: Configura a posição do background-image;
  • background-repeat: Configura a forma como a imagem de fundo é repetida. Os seguintes valores são possíveis: repeat, no-repeat, repeat-x, repeat-y, space e round;
  • background-size: Configura o tamanho da imagem de fundo no elemento. Pode-se especificar um tamanho, uma porcentagem ou usar um dos seguintes valores: auto, contain e cover
  • background-attachment: Configura a maneira como a imagem de fundo é fixada. Pode-se usar os seguintes valores: local, fixed e scroll.

Podemos usar uma shorthand para configurar essas propriedades sem ter que fazer diversas declarações. 

background: color image position repeat [size] attachment
