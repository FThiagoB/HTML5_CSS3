Usando listas

Existem basicamente dois tipos de listas, as listas numeradas, também chamada de lista ordenada, e as listas não ordenadas.

Listas ordenadas

São utilizadas para enumerar itens associando uma sequência ordenada a cada um deles. Especificando o parâmetro 'type' podemos mudar o tipo de numeração e usando o parâmetro 'start' mudamos o início da contagem.

  1. Primeiro item;
  2. Segundo item;
  3. Terceiro item;
  4. Quarto item;
  5. Quinto item;

  2. Segundo item;
  3. Terceiro item;
  4. Quarto item;
  5. Quinto item;

  2. Segundo item;
  3. Terceiro item;
  4. Quarto item;

Listas não ordenadas

Para as listas não ordenadas podemos especificar o parâmetro 'type' e definir o símbolo. Os valores podem ser: circle, square, disc ou none;

  • Primeiro item;
  • Segundo item;
  • Terceiro item;
  • Quarto item;

Lista de definição

Temos também listas de definições que permitem uma listagem de termos e descrições. Por exemplo:

Graus de liberdade
Número de variáveis independentes necessárias para a descrição completa da posição de um mecanismo.
Manipulador
Conjunto de elos conectados entre sí por meio de juntas. Dependendo da maneira como esses elos são conectados podemos ter um manipulador do tipo serial ou paralelo.

Pequeno exemplo

Podemos usar listas aninhadas que é bastante útil para certas aplicações. Usar CSS também permite adicionar outros efeitos para as listas. Por exemplo:

  • Desktop

    2. C/C++
    3. Java
  • Web
    • Front-End
      1. HTML5
      2. CSS3
      3. Javascript
    • Back-End
