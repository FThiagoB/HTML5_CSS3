Aprendendo a criar tabelas

Criamos uma tabela usando as tags <table> e <table>. Além disso, as seguinte tags são usadas para estruturar uma tabela:

  • Table Header - <th>: Indica uma célula de cabeçalho da tabela;
  • Table Row - <tr>: Linha da tabela;
  • Table Data - <td>: É a "coluna" da tabela, no entanto, dizemos que as linhas de uma tabela possuem dados.

O fechamento das tags tr e td são opcionais no HTML 5, mas é uma boa prática de organização manter os fechamentos.
